Job-Application MGT

Web API Core - 3.1, EF Core [Code First - EF Transaction] / SQL Server / Angular / React

Apply - Job

  • User can Create Job-Application by Applying to a Job [Initially Application Status = Applied]

User - Data

  • View User's Data By Resume-Created and Downloaded To User's Computer
  • View User's Data By Resume-Created and Emailed To User

Follow Up - Job

  • User can Follow Up Applied Job-Application
  • User can Filter Job-Applications based on
    • Contact-Person
    • Province / City
    • Applied On [Date-Range]
  • Application Status == (Any != [Closed])
      User can -->
    • [View / Edit / Delete] Job-Application
    • Upload (Connect) Resume File to a Job-Application
    • Download Resume File Connected to a Job-Application
    • Track Application Status by Date & Time wise
  • Application Status == [Closed]
      User can -->
    • [View / Delete] Job-Application
    • Download Resume File Connected to a Job-Application
    • Track Application Status by Date & Time wise

Resume - Creator

  • Create Personal Info [Add / Edit]
  • Create Technical Skills [Add / Edit / Remove]
  • Create Work Experience(s) [Add / Edit / Remove]
  • Create Education(s) [Add / Edit / Remove]
  • My - Resume
      User can -->
    • Preview Resume
    • Download Resume [Generate PDF Resume File and Download To User's Computer]
    • Email Resume [Generate PDF Resume File and Email as Attachment To User's Email Address]
